Watchly User Guide

Learn how to use your personal watchlist

Getting Started

Welcome to Watchly! This guide will help you manage your movies and TV shows effectively.

Adding Items

  • Click "+ Add Show" button
  • Enter the title, select status (To Watch/Watched/Upcoming)
  • Add genres separated by commas (e.g., Action, Drama, Comedy)
  • Optionally add date, release year, and notes
  • Check "Movie" if it's a movie instead of a TV show
  • Click "Add to Watchlist"

Managing Items

  • Mark Watched/Unwatched: Toggle completion status
  • Add Season/Movie: Track individual seasons or movie sequels
  • Edit: Modify title, genres, notes, or other details
  • Delete: Remove items with confirmation dialog

Search & Filter

  • Search: Type in the search box to find titles or genres
  • Status Filter: Show only Watched, Upcoming, or To Watch items
  • Genre Filter: Filter by specific genres

Data Management

  • Export: Download watchlist + rankings as JSON backup
  • Import: Upload previously exported JSON file
  • Reset All: Clear entire watchlist (with confirmation)
  • Auto Backup: Daily cloud backup (setup required)

Status Icons

  • ✓ - Watched
  • → - Upcoming
  • ○ - To Watch

Organization

Items are automatically organized alphabetically by first letter. Your stats show total items, watched count, and upcoming releases.

Leaderboard

  • Personal Rankings: Create your own ranked list of favorites
  • Add Items: Select from dropdown to add watchlist items to leaderboard
  • Drag & Drop: Reorder items by dragging them up/down
  • Auto Scroll: Page scrolls automatically when dragging near edges
  • Remove Items: Click × button with confirmation dialog
  • Sync: Rankings automatically sync with your main watchlist

Auto Backup Setup

Step 1: Get your free JSONBin.io API key

  • Go to jsonbin.io
  • Click "Create Account" (free)
  • Verify your email address
  • Go to "API Keys" section
  • Click "Create Access Key"
  • Copy your Master Key (starts with $2a$...)

Step 2: Enable Auto Backup

  • Go to Backup page in Watchly
  • Click "Setup API Key"
  • Paste your JSONBin.io Master Key
  • Click "Save & Test" to validate

How it works:

  • Backs up every 24 hours when changes are made
  • Uses only ~365 requests per year (out of 10,000 free)
  • Includes both watchlist and leaderboard data
  • Monitor status in real-time on Backup page

Backup Management

  • Status Monitoring: Real-time backup status and API usage
  • Force Backup: Manual backup trigger anytime
  • Test Connection: Verify API key is working
  • Account Info: View plan type and remaining requests
  • Manual Export/Import: Always available as backup option

Data Storage

  • Local Storage: Data saved in browser (can be lost)
  • Auto Backup: Cloud backup to JSONBin.io (recommended)
  • Manual Export: Download JSON files for safekeeping
  • Import: Restore from JSON files anytime
  • Combined Backup: Export includes both watchlist and rankings

Security

Use the "Lock" button to log out and secure your watchlist. You'll need to re-enter the password to access it again.
